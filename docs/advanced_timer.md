# Advanced Timer

**Module:** `timer`
**Class:** `AdvancedTimer`
**Category:** Automation

Time-windowed entity scheduler with enforcement rules. Supports three timer types: simple (on at time X), window (on between start/end times), and relative (on N minutes after event).
